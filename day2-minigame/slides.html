touch → 素早くファイル作成
cat → 中身を瞬時に確認

🛠️ 実務で使うディレクトリ操作

💻 開発現場の実例

1. プロジェクト構築

mkdir my-project → 新規プロジェクトフォルダ作成
cd my-project → プロジェクトディレクトリに移動
mkdir src docs tests → 標準的なフォルダ構造を一気に作成

2. ファイル管理の実務

touch README.md → プロジェクトの説明書を作成
echo "# My Project" > README.md → タイトルを書き込み
cat package.json → 設定ファイルの中身を確認

3. 効率的な移動術

cd ~ → ホームディレクトリに瞬間移動
cd .. → 一つ上の階層に移動
cd - → 直前の場所に戻る
