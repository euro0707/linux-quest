🔄 データ処理の現場例

1. ログ解析パイプライン

cat access.log | grep "ERROR" | wc -l → エラー件数の高速集計
tail -f app.log | grep "CRITICAL" → 重要エラーのリアルタイム監視
cat *.log | sort | uniq -c > summary.txt → 全ログの重複除去・集計・保存

2. データ分析処理

ps aux | grep python | awk '{print $2}' → Pythonプロセス一覧の抽出
ls -l | grep "^d" | wc -l → ディレクトリ数の高速カウント
find . -name "*.py" | xargs wc -l → 全Pythonファイルの行数集計

3. 自動化レポート生成

date > report.txt && ps aux >> report.txt → 日時付きシステムレポート生成
df -h | grep -v "tmpfs" | sort → ディスク使用量の整理表示
