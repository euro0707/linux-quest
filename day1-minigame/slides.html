📋 3つのコマンドの実用シーン

🏢 開発現場のリアル

1. echo の使い方

echo "Hello World" → 動作確認
echo $PATH → 環境設定確認
echo "処理完了" >> log.txt → 記録残し

2. pwd の使い方

pwd → 「今どこにいる？」の確認

• 作業ディレクトリの迷子防止

• パスをコピペで他の人に場所を伝える

3. ls の使い方

ls → 「ここに何がある？」
ls -l → 詳細情報（サイズ、日付、権限）
ls *.txt → 特定ファイルだけ表示
