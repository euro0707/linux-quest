🚀 実務で作るシステム管理ツール

🏢 エンタープライズレベルの統合例

1. 総合監視システム

#!/bin/bash - system_monitor.sh → システム全体の包括監視
ps aux | grep -v grep | wc -l → プロセス数監視
df -h | awk '{print $5}' | grep -o '[0-9]*' → ディスク使用率監視

2. 自動バックアップシステム

find /data -mtime -1 -type f → 最新ファイルの自動検出
tar -czf backup_$(date +%Y%m%d).tar.gz → 日付付き自動圧縮
chmod 600 *.tar.gz && chown admin:backup → セキュアなバックアップ権限設定

3. 運用レポート自動生成

echo "=== System Report $(date) ===" > report.txt → レポートヘッダー自動生成
uptime >> report.txt && free -h >> report.txt → システム情報の自動収集
